Файл истории версий программы Power Control
(записаны изменения с версии 2.06)

Значение символов:
+ Функция добавлена
- Функция или ошибка удалены (общая коррекция сбоев не приводится, чтобы иметь лучший обзор)
* Функция изменена
--------------------------------------------------------------------------------------------
Version 2.14
+ Added support for PSB 10000 series
+ Settings app: added support for new analog interface related settings (pin 9 and 10, only with PSB series)

Версия 2.13:
+ Добавлена функция Sandia в Мульти Контроль
+ Добавлены функции теста батареи "Статический заряд" и "Динамический тест" (доступно только для устройств PSB) в Генератор Функций и Мульти Контроль

Версия 2.12:
+ Мульти Контроль: добавлена функция симуляции солнечных массивов SAS (solar array simulation), которая расширяет функциональность стандартной PV функции
+ Генератор Функций: добавлена функция Sandia для PV тестирования

Версия 2.11:
* Исправлена проблема с обновлениями (всплывающее окошко касательно регистров со значением 0)
+ Добавлены кнопки вверх/вниз в ячейки ввода цифр, ручное увеличение/уменьшение кнопками стрелок также возможно
+ Добавлена конфигурация пинов 6, 14, 15 в приложение Настройки (доступно для серий PSI 9000, ELR 9000, EL 9000 B от KE 2.23 для пинов 6/14 и KE 2.24 для пина 15)
+ Добавлен ввод негативного значения в Мульти Контроль для серий PSB 9000 и PSBE 9000
+ Добавлена функция регистрации в приложение Генератор Функций
* Исправлена проблема с GUI с актуальным значением мощности выше чем 99999 Вт
* Исправлена проблема с обновлением прошивки HMI ведомых блоков

Версия 2.10:
+ Поддержка устройств серии PSB 9000 в приложении Генератор Функций
+ Раздельный обзор данных (графический) для функции DIN EN 50530 в режимах Day ET и Day UI
+ Расширенный демо режим добавлением блока PSB 9000
+ Версии программных прошивок теперь отображаются на верхней панели вместе с другой информации относительно устройства как модель и относительные значения
+ Оптимизировано окно Скрипта
+ Мульти Контроль, МРР слежение: добавлена табуляция с некоторыми опциями для активации повтора режимов МРР3 и МРР4 и паузы регистрации на часы и минуты во время длительных тестов

Версия 2.09:
+ Скриптовая функция: в приложении Терминал можно загрузить скрипт файл (формата CSV) для запуска в режиме пакетного процесса с регулировкой времени между командами
+ Поддержка серии PSB 9000
* Улучшены настройки CAN интерфейса
- Устранены ошибки касательно дисплея GUI и языков

Версия 2.08:
- Устранены ошибки при обновлении устройств в ЕАРС на Windows 10

Версия 2.07:
+ Поддержка для PV функции DIN EN 50530 в приложении Мульти Контроль
+ Поддержка генератора функций для серии EL 3000 B в приложении Генератор Функций
+ Поддержка новой серии ELR 9000 HP

Версия 2.06:
+ Настройка и контроль Генератора функций в приложении Мульти Контроль
+ Использование функции График в Мульти Контроль
+ Поддержка новой серии EL 3000 B (еще нет поддержки генератора функций)
+ Опция на пользование 14 дневной пробной лицензией (по запросу), которая разблокирует все лицензированные функции, но ограничивает приложение Мульти Контроль до 5 устройств
* Устранены ошибки при обновлении
* Улучшена функция PV, теперь она показывает MPP в визуализации таблицы PV
+ Расширена функция PV в соответствии с DIN EN 50530 (требуется обновление прошивок KE и HMI для серии PSI 9000)
